# Pokémon Data Analysis using Pandas

## 📌 Project Overview

This project demonstrates basic data analysis using the Pokémon dataset with the Pandas library in Python. The goal of this project is to practice data manipulation, exploration, and analysis techniques using Jupyter Notebook.

## 📂 Dataset

- **Dataset:** Pokemon.csv
- The dataset contains information about different Pokémon, including:
  - Name
  - Type
  - HP
  - Attack
  - Defense
  - Speed
  - Generation
  - Legendary status

## 🛠️ Technologies Used

- Python
- Pandas
- Jupyter Notebook

## 📊 Operations Performed

The following Pandas operations were performed:

- Loaded the dataset using `pd.read_csv()`
- Displayed the first and last few rows
- Checked dataset information using `info()`
- Generated summary statistics using `describe()`
- Selected specific rows and columns
- Filtered data based on conditions
- Sorted values
- Added and removed columns
- Performed basic indexing and slicing
